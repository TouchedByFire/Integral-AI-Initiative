# AI@Integral Initiative

React website for Integral University's AI@Integral Initiative portal.

## Overview

The AI@Integral Initiative portal serves as the central hub for AI-related programs, research, ethical guidelines, and multidisciplinary academic offerings at Integral University. The site is designed with a modern, responsive UI incorporating the university's branding.

## Scripts

- `npm run dev` starts the local development server.
- `npm run build` creates the production build in `dist`.
- `npm run preview` serves the production build locally.
